# Kampusin - Sistem Informasi Akademik

Dashboard akademik modern untuk mahasiswa dan dosen.

## Fitur

- Login mahasiswa & dosen
- Manajemen absensi
- Jadwal kuliah & ujian
- Pengumuman kampus
- Profil akademik
- UI responsif (mobile-first, auto-detect device)

- Zustand untuk state management
- Framer Motion untuk animasi
